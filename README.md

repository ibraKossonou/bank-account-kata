# Bank Account Kata

Ce projet est une simulation de comptes bancaires en Java.

Il permet de :

- Faire des dépôts et des retraits
- Afficher un relevé de compte avec la date, le montant et le solde
- Gérer deux types de comptes :
  *****Compte courant avec découvert autorisé
  **** Compte épargne avec calcul d'intérêts
  
 Les montants sont gérés avec "BigDecimal" pour une meilleure précision.
  
Ce projet utilise aussi les principes SOLID pour appliquer les meilleures pratiques.
